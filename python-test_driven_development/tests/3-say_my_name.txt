===============================
How to use 3-say_my_name.py
===============================

Import the function:
    >>> say_my_name = __import__('3-say_my_name').say_my_name

Normal usage:
    >>> say_my_name("John", "Smith")
    My name is John Smith

Only first name (note the trailing space):
    >>> say_my_name("Bob")
    My name is Bob
